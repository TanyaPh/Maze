Maze


Данная программа позволяет генерировать и отрисовывать идеальные лабиринты.

В программе предусмотрена кнопка для загрузки лабиринта из файла. Загруженный лабиринт отрисовывается на экране в поле размером 500 x 500 пикселей. Максимальный размер лабиринта - 50х50.

Предусмотрена возможноть автоматической генерации идеального лабиринта. Данный лабиринт создается согласно алгоритму Эллера Идеальным считается лабиринт, в котором из каждой точки можно попасть в любую другую точку ровно одним способом. Сгенерированный лабиринт не будет иметь изолированных областей и петель. Для генерации идеального лабиринта понадобится ввести только размерность лабиринта: количество строк и столбцов.

Предоставлена функция демонстрации решения лабиринта, который сейчас изображен на экране. Для использования пользователю необходимо задать начальные и конечные координаты искомого пути.
